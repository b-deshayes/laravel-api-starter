    Authentication

    APIs for users's authentication

    auth/login

    Get a JWT Token thought the given credentials.

    Example request:

    curl -X POST \
    "http://localhost:8081/api/v1/auth/login" \
    -H "Content-Type: application/json" \
    -H "Accept: application/json" \
    -d '{"email":"john@doe.com","password":"secret"}'

    const url = new URL(
    "http://localhost:8081/api/v1/auth/login"
);

let headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "email": "john@doe.com",
    "password": "secret"
}

fetch(url, {
    method: "POST",
    headers: headers,
    body: body
})
    .then(response => response.json())
    .then(json => console.log(json));
    
$client = new \GuzzleHttp\Client();
$response = $client->post(
    'http://localhost:8081/api/v1/auth/login',
    [
        'headers' => [
            'Content-Type' => 'application/json',
            'Accept' => 'application/json',
        ],
        'json' => [
            'email' => 'john@doe.com',
            'password' => 'secret',
        ],
    ]
);
$body = $response->getBody();
print_r(json_decode((string) $body));

    Example response (200):

    {
    "data": {
        "access_token": "e[...]7QOlDpr3d-tGUSTCefyjYN7tBIg",
        "token_type": "bearer",
        "expires_in": 3600
    }
}

    Example response (403):

    {
    "message": "api.v1.auth.login.invalid_credentials"
}

    HTTP Request

    POST api/v1/auth/login

    Body Parameters

    Parameter Type Status Description
    email string required The mail of the user.
    password string required The password of the user.
